Project 2

Command Line Software

Project Accomplishment

The main objective of this project is to facilitate building management and user organization. The software aims to track the occupancy of classrooms and provides a command-line interface for users.
The software utilizes a CRU format export that defines the class schedule for available rooms. It offers several options.
Users can search for rooms based on a specific class and find out the maximum capacity of a room. They can also check the availability of a desired room during the week and view the available rooms at a given time.
The software has been developed in JavaScript.

Project 4

DOCTOLIB

Project Accomplishment

The project involved building a website similar to Doctolib, aiming to manage all the information for scheduling appointments between patients and practitioners.
The website was developed using PHP.
The requested functionalities include the management of specialties, displaying practitioners and their specialties, counting the number of practitioners per patient, as well as various information presented in the form of tables.
This project also includes innovations on the website, such as a map displaying the practitioners closest to the user's location.
